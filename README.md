Allgemeine Informationen
========================

github guidelines
-----------------
- Nur sources committen, keine binaries

coding guidelines
-----------------

- Spaces statt Tabs
- Funktionsnamen klein
- Variablennamen klein
- Klassennamen groß
- Attributnamen klein
- Geschweifte Klammern:
	- bei Klassen/Methoden: neue Zeile
	- bei if, while etc.: gleiche Zeile
